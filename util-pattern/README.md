### 开闭原则
> 软件实体包括，项目中划分的模块，类和接口，方法
> 对扩展开放，对修改闭塞


### 里氏替换原则
> 里氏替换原则是实现开闭原则的重要方式之一
> 它克服了继承中重写父类造成的可复用性变差
> 子类可以扩展父类的功能，但不能改变父类原有的功能
